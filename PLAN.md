# Project Plan

此文件只保留已经讨论并确认、但尚未完成的项目计划。它不是开发日志，也不记录已完成事项；历史变更请查看 [CHANGELOG.md](CHANGELOG.md)。

## 已确认的待办

### 收获页 3D 场景与节点侧栏优化

按以下顺序实施，保持现有果实筛选、树冠缩放、适配、复位、阅读和重新生长行为不变：

1. **消除树根附近的圆形破碎色块**
   - 重点处理 `DagWorkbench` 为果树额外创建的圆柱形 `patch`；当前异常区域与该补丁的尺寸和位置一致，不属于 DAG 数据或果实节点。
   - 删除不必要的叠层，或改用不会与大草地表面产生深度冲突的单一地面实现；避免依赖接近共面的几何体、过大的场景尺度和 GPU 深度精度碰运气。
   - 在不同窗口尺寸、树冠大小、缩放距离和旋转角度下验证地面连续、颜色稳定，不再出现扇形、矩形碎片或闪烁。

2. **将果实点击聚焦改为固定镜头距离**
   - 移除当前 `max(220, currentDistance * 0.6)` 的比例缩放算法。
   - 以所选果实为观察目标，保留用户当前观察方向，但把相机放到统一的固定距离；为相机与节点重合、方向向量无效等情况提供稳定的默认方向。
   - 从树冠任意深度和任意初始缩放点击果实时，最终构图与果实可读大小应一致；动画时长遵循“减少动态效果”设置，适配与复位仍恢复全树视图。

3. **调整收获页三栏布局与树冠控制位置**
   - 从左侧筛选栏移除“果实”和“枝条”统计卡片；“全部”分类继续显示果实总数，枝条数量不再作为用户界面的主要信息。
   - 将“树冠大小”滑杆移到 3D 画布顶部中央，以轻量悬浮工具条覆盖在模型上方；保留当前数值、键盘调节、焦点样式和无障碍标签，不得因悬浮层遮挡果实点击、错误提示或画布旋转操作。
   - 左侧栏仅保留收获标题、说明、状态筛选和更新时间，并在移除统计与滑杆后缩窄为紧凑宽度；同步把释放出的空间交还给 3D 画布。
   - 窄窗口下让树冠控制条安全换行或贴近画布上沿，避免与画布操作提示、适配/复位按钮及右侧检查器重叠；没有进入 3D 果树状态时不显示无意义的悬浮控制条。

4. **为果实状态和生长阶段提供即时说明**
   - 左侧每个果实状态筛选项在鼠标悬停时显示该状态的简要含义，并复用现有 `ripe.*.desc` 中英文文案；“全部”补充“显示所有果实”的说明。
   - 生长页的每个阶段整行均可触发说明，不再只依赖阶段名称上的浏览器原生 `title`。将现有 `stage.*.tip` 改为更明确的功能说明，解释该阶段正在处理什么，例如文本提取、内容清洗、教学单元切分、向量化、知识节点聚合、先修关系构建和知识文件生成。
   - 使用统一的应用内提示层，鼠标悬停后即时出现，并支持键盘聚焦、`aria-describedby` 和清晰的焦点状态；提示层不引发布局跳动、不遮挡当前操作，并在离开、失焦或按下 Escape 后关闭。
   - 对无悬停能力的设备保留可聚焦或可点击的说明入口，避免核心状态含义只能通过鼠标获得。

5. **重构右侧节点检查器的信息层级**
   - 将成熟度、节点编号、标题和推荐说明整理成紧凑的头部区域，减少当前连续散排文本造成的视觉噪声和大片无效留白。
   - 把“只看后续果实”、未读先修提醒、先修、后继、定义和成品操作划分为清晰的内容区块；统一区块间距、标题层级、标签和节点按钮样式，并为无先修或无后继提供明确空状态。
   - 突出“开始学习/阅读”主操作，弱化辅助元数据；保留长标题、长定义和大量关系项的换行、滚动与键盘焦点可用性。
   - 当前工作区已经移除固定为 `default` 的“来源”区块，后续侧栏重构不得重新引入来源目录信息。
   - 同步核对窄窗口布局，避免侧栏变为纵向内容后出现溢出、顺序混乱或画布高度异常。

6. **让推荐理由使用可国际化的结构化契约**
   - 后端不再直接向 GUI 返回 `Root node; ready to start.`、`All prerequisite nodes have been read.` 等英文展示文本，改为稳定的推荐理由代码（及必要参数）。
   - 前端通过现有 i18n 字典渲染中英文文案；为旧缓存或旧 API 数据保留安全回退，且不得把未知英文原文直接泄漏到中文界面。
   - 一并核对节点检查器中的动态状态、空状态、错误与推荐文案，确保中文模式没有遗留英文展示文本。

7. **补充回归验证**
   - 为固定聚焦算法增加单元测试，断言不同初始相机位置和不同节点位置最终都满足相同距离，并覆盖退化方向。
   - 为推荐理由代码、API 序列化和中英文映射增加测试，覆盖根节点、先修已读和非推荐节点。
   - 增加收获页交互与视觉检查：根节点/非根节点选择、连续切换果实、筛选后聚焦、适配、复位、悬浮树冠控制、窗口缩放及不同设备像素比。
   - 覆盖果实筛选和生长阶段提示层的鼠标悬停、键盘聚焦、Escape 关闭、边界碰撞与中英文文案，确认提示层不会导致页面跳动或遮挡关键控件。

验收标准：3D 地面在各观察角度下无圆形破碎色块或闪烁；点击任意果实都会到达统一镜头距离；树冠滑杆稳定悬浮在画布顶部且左侧栏明显缩窄；果实状态和生长阶段的含义可通过鼠标与键盘即时获知；中文界面不再出现英文推荐理由；右侧侧栏信息分组清晰、主操作突出，并且不影响现有收获流程。
