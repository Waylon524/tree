# 02. 平衡常数与标准自由能变化

## 核心内容

### 1. 标准自由能变与平衡常数的关系

#### 1.1 范特霍夫等温式

对于任意气相反应：
\[ aA(g)+bB(g)\rightleftharpoons cC(g)+dD(g) \]
恒温条件下，反应的摩尔吉布斯自由能变 ΔG 与标准自由能变 ΔG° 及反应商 Q 的关系为：

\[ \Delta G = \Delta G^\circ + RT\ln Q \quad (\text{范特霍夫等温式}) \]

式中：
- \(R = 8.314\ \text{J·mol}^{-1}\text{·K}^{-1}\)（摩尔气体常数），
- \(T\) 为绝对温度（K），
- \(Q\) 为反应商，对气体反应其无量纲形式为 \(Q^\circ = \dfrac{(P_C/p^\circ)^c (P_D/p^\circ)^d}{(P_A/p^\circ)^a (P_B/p^\circ)^b}\)（详见下文 1.3）。

当体系达到化学平衡时，\(\Delta G = 0\)，且此时反应商等于标准平衡常数 \(K^\circ\)，代入得：

\[ 0 = \Delta G^\circ + RT\ln K^\circ \]

移项即得核心关系式：

\[ \boxed{\Delta G^\circ = -RT\ln K^\circ} \]

此式连通了热力学函数（ΔG°）与平衡位置（K°）。反之也可写成：

\[ \ln K^\circ = -\frac{\Delta G^\circ}{RT} \quad \text{或} \quad \lg K^\circ = -\frac{\Delta G^\circ}{2.303RT} \]

#### 1.2 单位与数值换算——关键警示

- **ΔG° 的单位必须统一为 J·mol⁻¹**：热力学数据表中 ΔG° 常以 kJ·mol⁻¹ 给出。计算时必须乘以 1000 转换为 J·mol⁻¹，否则结果会产生 1000 倍的巨大误差。
  - 例如：ΔG° = –11.7 kJ·mol⁻¹ → 代入公式前应写为 –11700 J·mol⁻¹。
  
- **R 的单位固定为 J·mol⁻¹·K⁻¹**：无需转换。
  
- **对数必须使用自然对数 ln**：常见计算器上若只有常用对数 lg（以 10 为底），需使用换算：
  \[ \ln x = 2.3026\ \lg x \]
  因此公式也可写为：
  \[ \lg K^\circ = -\frac{\Delta G^\circ}{2.303\,RT} \]
  注意系数 2.303 ≈ ln 10。

- **有效数字**：ΔG° 一般给出 3~4 位有效数字，计算 ln K° 时中间结果可多保留几位，最终 K° 保留与数据相同的有效数字位数。

#### 1.3 标准平衡常数 K° 的无量纲性

与经验平衡常数 K_P 不同，**标准平衡常数 K° 是一个无量纲的纯数**。原因在于，K° 的定义涉及**活度**（无量纲量）：
\[ K^\circ = \frac{(a_C)^c (a_D)^d}{(a_A)^a (a_B)^b} \]
对于理想气体，活度 = 分压除以标准压力 \(p^\circ\)：
\[ a_i = \frac{P_i}{p^\circ},\quad p^\circ = 1\ \text{bar} \]
因此：
\[ K^\circ = \frac{(P_C/p^\circ)^c (P_D/p^\circ)^d}{(P_A/p^\circ)^a (P_B/p^\circ)^b} = K_P \cdot (p^\circ)^{-\Delta n} \]
其中 \(\Delta n = (c+d)-(a+b)\)。

**要点**：
- 在计算 Q° 或 K° 时，**必须先将每个分压除以 p°（1 bar）**，得到无量纲比值后再代入表达式。若分压单位不是 bar，需先换算（1 bar = 100 kPa ≈ 0.987 atm）。
- 经验平衡常数 K_P 可能带有单位（当 Δn ≠ 0 时），而 K° 永远无量纲。两者数值仅在 Δn = 0 时相等（且此时 p° 因子为 1）。

**示例**：对于光气分解反应 \(\mathrm{COCl_2(g)\rightleftharpoons CO(g)+Cl_2(g)}\)，Δn = 1。若已知 K_P = 599.8 kPa，则：
\[ K^\circ = K_P \cdot (p^\circ)^{-1} = 599.8\ \text{kPa} \times \frac{1}{100\ \text{kPa}} = 5.998 \]
（因为 1 bar = 100 kPa）

#### 1.4 ΔG° 的估算判据

虽然ΔG° 是标准状态下的自由能变，但可以粗略判断反应完成的限度：
- ΔG° < –40 kJ·mol⁻¹：反应倾向强烈，可认为单向进行；
- ΔG° > +40 kJ·mol⁻¹：平衡强烈向左，生成物极少；
- 介于两者之间：反应方向取决于具体条件，需结合 Q 与 K° 比较。

### 2. 反应商与自由能变的关系

将 ΔG° = –RT ln K° 代入范特霍夫等温式，可得另一常用形式：
\[ \Delta G = -RT\ln K^\circ + RT\ln Q = RT\ln\frac{Q}{K^\circ} \]
由此直接得到方向判据：
- 若 \(Q < K^\circ\)，则 \(\ln(Q/K^\circ) < 0\)，\(\Delta G < 0\) → 反应向右自发进行；
- 若 \(Q > K^\circ\)，则 \(\ln(Q/K^\circ) > 0\)，\(\Delta G > 0\) → 反应向左自发进行；
- 若 \(Q = K^\circ\)，则 \(\Delta G = 0\) → 平衡。

**注意**：此处的 Q 也必须采用无量纲形式 \(Q^\circ\)（即分压除以 p° 后代入）。

## 例题

**题目**：已知反应 \(\mathrm{2NO_2(g) \rightleftharpoons N_2O_4(g)}\) 在 298 K 时的标准自由能变 ΔG° = –4.73 kJ·mol⁻¹。计算该温度下反应的 K° 值，并判断当体系中 NO₂ 分压为 0.20 bar、N₂O₄ 分压为 0.10 bar 时反应的自发方向。

**解**：

1. **单位转换**：
   \[ \Delta G^\circ = -4.73\ \text{kJ·mol}^{-1} = -4730\ \text{J·mol}^{-1} \]

2. **计算 K°**：
   \[ \ln K^\circ = -\frac{\Delta G^\circ}{RT} = -\frac{(-4730)}{8.314 \times 298} = \frac{4730}{2477.572} \approx 1.909 \]
   \[ K^\circ = e^{1.909} \approx 6.75 \]

   （若使用常用对数：\(\lg K^\circ = \dfrac{4730}{2.3026 \times 8.314 \times 298} = \dfrac{4730}{5706.8} \approx 0.8289\)，则 \(K^\circ = 10^{0.8289} \approx 6.75\)，结果一致。）

3. **计算反应商 Q°**：
   反应：\(2\mathrm{NO_2} \rightleftharpoons \mathrm{N_2O_4}\)
   \[ Q^\circ = \frac{P_{\mathrm{N_2O_4}}/p^\circ}{(P_{\mathrm{NO_2}}/p^\circ)^2} = \frac{0.10/1}{(0.20/1)^2} = \frac{0.10}{0.04} = 2.5 \]

4. **比较 Q° 与 K°**：
   \(Q^\circ = 2.5 < 6.75 = K^\circ\) → 反应向右（生成 N₂O₄）自发进行。

5. **验证 ΔG**：
   \[ \Delta G = RT\ln\frac{Q}{K^\circ} = 8.314 \times 298 \times \ln\frac{2.5}{6.75} \approx 2477.6 \times (-0.9927) \approx -2460\ \text{J·mol}^{-1} < 0 \]
   方向判据一致。

**答案**：\(K^\circ = 6.75\)（无量纲）；反应向右进行。

## 常见误区

1. **单位陷阱**：ΔG° 以 kJ 给出时，忘记乘以 1000 转换为 J。直接代入将导致 ln K° 缩小 1000 倍，K° 严重偏离正确值。
   - ✅ 正确做法：ΔG° (kJ·mol⁻¹) × 1000 → J·mol⁻¹。

2. **底数混淆**：使用 lg 而非 ln 时未乘转换因子 2.303。
   - 公式：\(\ln x = 2.3026\ \lg x\)；若直接写 \(\lg K^\circ = -\Delta G^\circ/(RT)\) 会差 2.303 倍。
   - ✅ 正确做法：\(\lg K^\circ = -\Delta G^\circ/(2.303 RT)\)。

3. **忘记除以标准压力**：计算 Q° 或 K° 时直接使用分压数值（例如 0.20 bar 写成 0.20），而不是先除以 p°（1 bar）。当分压单位不为 bar 时错误更严重。
   - ✅ 正确做法：\(a_i = P_i / p^\circ\)，且 p° = 1 bar。

4. **混淆 K° 与 K_P**：误以为 K_P 可直接代入 ΔG° = –RT ln K_P。实际上 ΔG° 对应的必须是无量纲的 K°，两者通过 \(K^\circ = K_P \cdot (p^\circ)^{-\Delta n}\) 换算。
   - ✅ 正确做法：先换算为 K° 再使用公式。

5. **方向判据遗忘**：记住 ΔG 与 Q/K 的关系：\(\Delta G < 0 \Leftrightarrow Q < K\)，\(\Delta G = 0 \Leftrightarrow Q = K\)，\(\Delta G > 0 \Leftrightarrow Q > K\)。

## 自测题

**1. 基础计算**：已知反应 \(\mathrm{CO(g) + H_2O(g) \rightleftharpoons CO_2(g) + H_2(g)}\) 在 298 K 时的 ΔG° = –28.6 kJ·mol⁻¹。计算该反应的 K° 值，并判断：若起始时各组分分压均为 0.50 bar，反应向哪方向进行？

**2. 逆向计算**：反应 \(\mathrm{N_2(g) + 3H_2(g) \rightleftharpoons 2NH_3(g)}\) 在 500 K 时测得 \(K_P = 1.09 \times 10^{-10}\ \text{kPa}^{-2}\)。计算该温度下的标准平衡常数 K° 和 ΔG°（注意单位换算，p° = 100 kPa）。已知 R = 8.314 J·mol⁻¹·K⁻¹。

**3. 综合应用**：反应 \(\mathrm{2NO(g) + O_2(g) \rightleftharpoons 2NO_2(g)}\) 在 298 K 时的 K° = 1.56 × 10⁶。某一容器中各气体分压分别为：NO = 0.010 bar，O₂ = 0.010 bar，NO₂ = 0.010 bar。通过计算 ΔG 判断反应方向，并计算该条件下反应的 ΔG（单位 J·mol⁻¹）。

（提示：1 bar = 100 kPa；所有分压先除以 p° 得到无量纲比值；常用对数与自然对数可选用。）
