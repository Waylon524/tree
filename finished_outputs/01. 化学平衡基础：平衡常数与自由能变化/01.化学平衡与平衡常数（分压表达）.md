# 01. 化学平衡与平衡常数（分压表达）

## 学习目标与先修前置

**学习目标**
- 理解化学平衡的动态本质：正逆反应速率相等，各组分分压保持不变。
- 掌握气相反应平衡常数 $K_P$ 的定义、书写规则及物理意义。
- 能根据初始分压与平衡条件（总压或解离度）计算平衡分压和 $K_P$。
- 区分反应商 $Q$ 与 $K_P$，并利用比较结果判断反应方向。
- 应用勒夏特列原理解释压强/体积改变对平衡的影响。

**先修前置**
- 理想气体状态方程：$PV = nRT$。
- 道尔顿分压定律：混合气体总压等于各组分分压之和，分压与摩尔分数成正比。

## 核心内容

### 1. 化学平衡的基本概念

可逆反应在同一条件下既能正向进行又能逆向进行。当**正反应速率等于逆反应速率**时，反应物和生成物的浓度（或分压）不再随时间改变，此时体系处于**化学平衡**状态。化学平衡是动态平衡，反应并未停止，只是净变化为零。

### 2. 平衡常数 $K_P$ 的定义与书写

对于任意气相反应：
$$ aA(g) + bB(g) \rightleftharpoons cC(g) + dD(g) $$
在给定温度下达到平衡时，各组分分压满足：
$$ K_P = \frac{(P_C)^c \; (P_D)^d}{(P_A)^a \; (P_B)^b} $$
其中 $P_i$ 为组分 $i$ 的平衡分压（单位常为 kPa 或 bar）。$K_P$ 仅与温度有关，与总压、初始浓度无关。注意：
- 纯固体、纯液体的浓度视为常数，不写入平衡常数表达式（本单元不涉及，但后续章节会展开）。
- 经验平衡常数 $K_P$ 通常具有单位，单位由 $\Delta n = (c+d) - (a+b)$ 决定：若 $\Delta n = 0$，$K_P$ 无量纲；若 $\Delta n \neq 0$，$K_P$ 有单位为压力单位的 $\Delta n$ 次幂。

### 3. 反应商 $Q$ 与平衡移动方向

定义**反应商** $Q_P$ 为任意时刻（非平衡态）的分压幂乘积之比，形式与 $K_P$ 完全相同：
$$ Q_P = \frac{(P_C)^c \; (P_D)^d}{(P_A)^a \; (P_B)^b} $$
比较 $Q_P$ 与 $K_P$ 的大小可判断反应方向：
- $Q_P < K_P$：反应向右（正方向）进行，使 $Q_P$ 增大直至等于 $K_P$。
- $Q_P > K_P$：反应向左（逆方向）进行，使 $Q_P$ 减小直至等于 $K_P$。
- $Q_P = K_P$：体系处于平衡。

此判断方法基于范特霍夫等温式（本单元不推导，仅给出结论）。

### 4. 平衡分压的推算方法——设转化量法

在恒温恒容条件下（$V$ 和 $T$ 不变），分压与物质的量成正比，因此分压的变化量直接反映反应进度。

**步骤**：
1. 写出反应计量方程式并标出各物质初始分压（或由初始物质的量、摩尔分数换算）。
2. 设某组分的分压变化量为 $x$（根据反应式确定正负号），其他组分的变化量按化学计量数倍数写出。
3. 列出平衡分压表达式：初始分压 ± 变化量。
4. 利用给定条件（总压、解离度、$K_P$ 值等）建立方程，求解 $x$。
5. 代回求各平衡分压，并计算 $K_P$。

**核心关系**：对于反应 $aA + bB \rightleftharpoons cC + dD$，设 $\Delta P_A = y$（$y>0$ 表示减少），则 $\Delta P_B = \frac{b}{a}y$，$\Delta P_C = -\frac{c}{a}y$（若 $\Delta P_A$ 为减少，则生成物分压增加，符号为负），总压变化 $\Delta P_{\text{总}} = \Delta P_A + \Delta P_B + \Delta P_C + \Delta P_D = \left(\frac{c+d-a-b}{a}\right)y$。

### 5. 压强（体积）变化对平衡的影响——勒夏特列原理

若体系温度不变，改变总压（通过改变体积）会引起各组分分压按相同比例变化，从而改变 $Q_P$：

- **增大总压（压缩体积）**：所有分压同比例增大。设 $\Delta n = (c+d)-(a+b)$，则 $Q_P = 2^{\Delta n} K_P$。若 $\Delta n > 0$（气体分子数增加），$Q_P > K_P$，平衡逆向移动；若 $\Delta n < 0$，$Q_P < K_P$，平衡正向移动；若 $\Delta n = 0$，$Q_P = K_P$，平衡不移动。
- **减小总压（扩大体积）**：所有分压同比例减小，$Q_P = (1/2)^{\Delta n} K_P$，平衡移动方向与上述相反。

此规律与勒夏特列原理一致：平衡总是向削弱压强改变的方向移动。

## 例题

**题目**：光气（$\mathrm{COCl_2}$）高温分解：$\mathrm{COCl_2(g) \rightleftharpoons CO(g) + Cl_2(g)}$。在 900 K，一体积恒定的容器中，一定量 $\mathrm{COCl_2}$ 的初始压强为 101.3 kPa。平衡后测得体系总压强为 189.6 kPa。计算该反应的平衡常数 $K_P$。

**解**：

1. 写出反应式，设 $\mathrm{COCl_2}$ 的分压减少 $x$ kPa（因为反应正向进行，生成物增加）：
   $$ \mathrm{COCl_2(g)} \rightleftharpoons \mathrm{CO(g)} + \mathrm{Cl_2(g)} $$
   初始分压 (kPa)：101.3   0   0
   变化分压 (kPa)： $-x$   $+x$   $+x$
   平衡分压 (kPa)：101.3$-x$   $x$   $x$

2. 总压关系：
   $$ (101.3 - x) + x + x = 101.3 + x = 189.6 $$
   解得 $x = 88.3 \text{ kPa}$。

3. 各平衡分压：
   $$ P_{\mathrm{COCl_2}} = 101.3 - 88.3 = 13.0 \text{ kPa} $$
   $$ P_{\mathrm{CO}} = 88.3 \text{ kPa} $$
   $$ P_{\mathrm{Cl_2}} = 88.3 \text{ kPa} $$

4. 平衡常数：
   $$ K_P = \frac{P_{\mathrm{CO}} \cdot P_{\mathrm{Cl_2}}}{P_{\mathrm{COCl_2}}} = \frac{88.3 \times 88.3}{13.0} = 599.8 \text{ kPa} $$

   （单位：$\Delta n = 1$，因此 $K_P$ 具有压力单位。）

## 常见误区

1. **忽视反应计量系数**：平衡常数表达式中幂指数必须与计量系数严格一致，不能写成 $P_C^c$ 错写成 $c \cdot P_C$。
2. **设未知数时未按系数比例**：例如对于 $\mathrm{N_2 + 3H_2 \rightleftharpoons 2NH_3}$，若设 $\mathrm{N_2}$ 分压减少 $y$，则 $\mathrm{H_2}$ 减少 $3y$，$\mathrm{NH_3}$ 增加 $2y$，切忌写成等量变化。
3. **混淆 $Q$ 与 $K$ 的使用条件**：$Q$ 用于任意时刻，$K$ 仅用于平衡态；只有平衡时才能将分压代入 $K_P$ 表达式。
4. **忽略总压与分压的关系**：在恒容容器中，分压变化总和等于总压变化值。若总压已知，可由此列方程，避免重复假设。
